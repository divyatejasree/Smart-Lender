# Tools and Technologies Used

## Smart Lender - Loan Approval Prediction System

The Smart Lender project was developed using Python-based tools and libraries for data processing, machine learning, model evaluation, and web application deployment.

## Anaconda Navigator

Anaconda Navigator is a desktop graphical interface used to manage Python environments and data science packages. It helps install and launch tools such as Jupyter Notebook, Spyder, and other data science applications.

Official link: https://www.anaconda.com/download

## PyCharm / Visual Studio Code

PyCharm is a Python IDE that provides code editing, debugging, and project management features. Visual Studio Code was also used for writing, editing, and running the Smart Lender Flask application.

Official links:

- https://www.jetbrains.com/pycharm/
- https://code.visualstudio.com/

## NumPy

NumPy is a Python library used for numerical computing. It supports arrays, mathematical operations, and numerical processing required during machine learning model development.

Official link: https://numpy.org/doc/stable/

## Pandas

Pandas is used for data manipulation and analysis. In this project, it is used to load applicant data, process structured loan application records, and prepare the dataset for model training.

Official link: https://pandas.pydata.org/docs/

## Scikit-learn

Scikit-learn is a machine learning library used to build and evaluate classification models. Smart Lender uses algorithms such as Decision Tree, Random Forest, K-Nearest Neighbors, and preprocessing pipelines.

Official link: https://scikit-learn.org/stable/

## Matplotlib

Matplotlib is a visualization library used to create charts and graphs for analyzing dataset patterns, model performance, and feature relationships.

Official link: https://matplotlib.org/stable/

## Seaborn

Seaborn is a statistical data visualization library built on top of Matplotlib. It is useful for creating attractive graphs such as heatmaps, count plots, and correlation charts.

Official link: https://seaborn.pydata.org/

## Flask

Flask is a lightweight Python web framework used to deploy the trained machine learning model as a web application. In Smart Lender, Flask handles the web form, receives applicant input, and displays the loan approval prediction result.

Official link: https://flask.palletsprojects.com/

## Summary

These tools and libraries together support the complete Smart Lender workflow: data preprocessing, model training, model evaluation, and real-time web-based prediction.
